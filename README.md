## nodejs-NAPI-demo

A demo for learning nodejs NAPI(including NAN)

## How to run

### Run for NAN:

```
cd packages/md5-NAN
yarn/npm install
npm run install
node test/index.js
```

### Run for NAPI

```
cd packages/md5-NAPI
yarn/npm install
npm run install
node test/index.js
```
